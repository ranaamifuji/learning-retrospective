# 学習の振り返り

バイブコーディング学習の成果・共通ナレッジ・ステップ別学習をまとめた公開ノートです。

## 公開URL

GitHub Pages 有効化後:

https://ranaamifuji.github.io/learning-retrospective/

## 使い方

1. ブラウザで `index.html` を開く
2. カテゴリー（成果物／共通ナレッジ／ステップ別学習）で絞り込む
3. キーワード検索で用語やつまずきを探す

## 含めるもの / 含めないもの

| 含める | 含めない |
|--------|----------|
| 日付ごとの成果 | 日次の詳細ログ（ローカル専用） |
| 用語・Git・指示のコツ | パスワード、認証コード、APIキー |
| つまずき事例（症状→原因→解決） | 実メールアドレス |

## ローカルでの編集場所

学習ワークスペース内の `学習の振り返り/` を編集し、このリポジトリへ反映します。
