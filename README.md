# 学習の振り返り

バイブコーディング学習の成果・初心者FAQ・共通ナレッジ・ステップ別学習をまとめた公開ノートです。

## 公開URL

GitHub Pages 有効化後:

https://ranaamifuji.github.io/learning-retrospective/

## 使い方

1. ブラウザで `index.html` を開く
2. カテゴリー（成果物／初心者FAQ／共通ナレッジ／ステップ別学習）で絞り込む
3. キーワード検索で質問・回答・用語・つまずきを探す
4. FAQの質問を開き、回答→理由→確認手順→成功の目印の順で読む

## 初心者FAQの方針

- 実際にエージェントへ確認して解決した内容だけを載せる
- 「問・回答・理由・確認手順・成功の目印・専門用語」を基本の1組にする
- 日次ログをそのまま転載せず、個人情報を除いて初心者向けに再編集する
- 画面キャプチャは文章だけでは操作場所や状態を判断しにくい場合だけ使う
- 画像を使う場合は `assets/faq/` に置き、代替テキストと説明文を付ける
- 秘密情報が写った画像は加工して使わず、安全な状態で撮り直す

## 含めるもの / 含めないもの

| 含める | 含めない |
|--------|----------|
| 日付ごとの成果 | 日次の詳細ログ（ローカル専用） |
| 用語・Git・指示のコツ | パスワード、認証コード、APIキー |
| 初心者FAQ（問→回答→理由） | 実メールアドレス |
| 匿名化した安全な画面キャプチャ | 個人パス・通知・秘密情報が写った画像 |

## ローカルでの編集場所

学習ワークスペース内の `学習の振り返り/` を編集し、このリポジトリへ反映します。
